# San-Andreas-Booking-System
San Andreas Booking System is a WIP LSPDFR/RAGEPluginHook booking and processing plugin.

Current features:
- RAGEPluginHook plugin loads successfully
- RAGENativeUI booking menu works
- Cell selection menu prototype
- Suspect info menu prototype
- Search results menu prototype
- Evidence submission status system
- Fingerprint scan status system
- Report generation status system
- Jail finalization status system

Goal:
The goal is to integrate with Policing Redefined so suspect ID/name check data and search result data can be imported into the booking system.
